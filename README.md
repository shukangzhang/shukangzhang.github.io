⌈人在写作时，总是孤身一人，作品实际上是个人的独白，是一些发出的信。我觉得自己太缺少与人交流的机会---我相信，这是写严肃文学的人共同的体会。但是这个世界上除了有自己，还有别人，除了身边的人，还有整个人类。写作的意义，就在于与人交流。因为这个缘故，我一直在写。⌋
